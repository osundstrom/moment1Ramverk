Publicering - Moment 1

Jag valde att publicera med vercel.

Vercel har stöd för Next.js, SvelteKit, Astro, React, Gatsby, Nuxt, Remix, Vite och CRA.

Det är gratis att använda för mig som student/egna projekt men har även betallösningar för lösningar till företag eller större arbeten.

Vercel är väldigt enkelt att publicera till. Då jag tidigare använt Netlify så ser publicerings processen nästan exakt likadan ut, mitt Github konto kopplas till sidan och sedan kan jag publicera repositorit direkt behövde inte ens ändra några inställningar då det automatisk kör npm run build och npm install. Det ända jag behövede välja var att publicera main branch.
